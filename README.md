# Analizador Cuántico – Proyecto TFG

Proyecto mixto R + Python para la monitorización y análisis de datos en redes QKD.

## Estructura del proyecto
- `data/raw`: datos originales
- `data/processed`: datos limpios generados con R
- `src/`: scripts R y Python
- `notebooks/`: notebooks de desarrollo
- `reports/`: gráficos y resultados exportados

## Tecnologías
- R (Data analisis)
- Python (ML)
- Isolation Forest & Autoencoders
- VSCode + RStudio
